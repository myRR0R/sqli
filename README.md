# Eugene Muravitskiy

### Install [Sinatra](http://www.sinatrarb.com/)
```bash
gem install sinatra
```

### Update `npm` [npm](https://www.npmjs.com/)
```bash
npm install
```

### Run it:
```bash
# Just server:
ruby main.rb

# All together:
npm start
```
### And Open your browser:
Accessibe by http://localhost:4567/
